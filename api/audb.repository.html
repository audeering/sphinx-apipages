repository()¶

audb.repository(name, version)[source]¶

Return repository that stores the requested database.

If the database is stored in several repositories, only the first one is returned. The order of the repositories to look for the database is given by config.REPOSITORIES.

Parameters

  • name (str) – database name

  • version (str) – version string

Return type

Repository

Returns

repository that contains the database

Raises

RuntimeError – if database or version is not found

Examples

>>> audb.repository('emodb', '1.4.1')
Repository('data-public', 'https://audeering.jfrog.io/artifactory', 'artifactory')
